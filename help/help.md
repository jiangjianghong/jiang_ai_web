### 使用方法

1. 打开[Notion](https://www.notion.so/)并登录你的账户。如果你还没有账户，可以免费注册一个, 建议使用桌面app, 这个软件完全免费，千万不要付费下载。

2. 创建一个数据库，点击左侧的 "+新增" 按钮，然后选择新建数据库。
![alt text](image-2.png)

3. 编辑数据库。

(1)创建名称属性
默认是带有名称数据的，不需要改变它
![alt text](image-3.png)
(2)创建网址属性
点击 "+添加属性" 按钮，选择 "网址" 视图，然后点击 "创建"。
![alt text](image-4.png)
(3) 创建账号属性
点击 "+添加属性" 按钮，选择 "文本" 视图，然后点击 "创建"。
创建完，再点击刚刚创建的文本属性，重命名为 "账号"。
![alt text](image-5.png)
(4) 创建密码属性
点击 "+添加属性" 按钮，选择 "文本" 视图，然后点击 "创建"。
创建完，再点击刚刚创建的文本属性，重命名为 "密码"。
![alt text](image-6.png)
(5) 创建选择属性
点击 "+添加属性" 按钮，选择 "选择" 视图，然后点击 "创建"。
创建完，再点击刚刚创建的选择属性，重命名为 "Select"。
![alt text](image-7.png)
(6) 创建描述属性
点击 "+添加属性" 按钮，选择 "文本" 视图，然后点击 "创建"。
创建完，再点击刚刚创建的文本属性，重命名为 "描述"。

4. 获取notion集成令牌
(1) 打开[Notion集成页面](https://www.notion.so/my-integrations)，登录后，点击 "新集成" 按钮。
![alt text](image-8.png)
(2) 填写集成信息，名称可以随意填写，选择你的工作区(默认就一个)，类型就是内部，logo随意，然后点击 "提交" 按钮。
(3) 创建成功后，点击配置集成设置。
**秘钥**
里面有一个内部集成秘钥
点击"显示",复制这个秘钥，稍后会用到。
![alt text](image-9.png)

5. 获取Notion Web Clipper扩展程序
(1) 打开[Notion Web Clipper扩展程序页面](https://chromewebstore.google.com/detail/notion-web-clipper/knheggckgoiihginacbkhaalnibhilkk)，根据你的浏览器，选择对应的扩展程序安装。
(2) 安装完成后，点击浏览器右上角的Notion图标，登录你的Notion账户，授权扩展程序访问你的Notion数据。

6. 添加你的第一个条目
随便打开一个你要收藏的网页。
点击浏览器右上角的Notion图标，选择你刚刚创建的数据库，然后填写相关信息，点击 "保存" 按钮。
![alt text](image-10.png)

如果右上角没有Notion图标，点击浏览器的扩展程序图标，找到Notion Web Clipper，点击它即可。

如果右上角没有扩展程序图标，点击浏览器右上角的三个点，打开设置，启用扩展和扩展程序图标显示，不会可以网络搜索。

回到你的Notion工作空间，刷新页面，你会看到你刚刚添加的条目已经出现在数据库中。

7. 将秘钥添加到工作空间
回到我们的网站，点击左上角的工作空间。再点击工作空间右上角设置图标，打开设置页面。
**秘钥**
把你刚刚复制的Notion集成秘钥，粘贴到 "Notion API秘钥" 输入框中
**数据库ID**
回到Notion，打开你创建的数据库页面，
如果你使用的是网页版，直接复制浏览器地址栏中的URL。
如果你使用的是桌面版，点击右上角的 "分享" 按钮，选择 "复制链接"。
得到的链接类似于：
```net
https://www.notion.so/aaaaaaaaaaaaaa?v=bbbbbbbbbb
```
或者
```net
https://www.notion.so/aaaaaaaaaaaaaaa?v=bbbbbbbbbbbb&source=copy_link
```
其中，`aaaaaaaaaaaaaa` 或 `aaaaaaaaaaaaaaa` 就是你的数据库ID。
把它粘贴到 "Notion数据库ID" 输入框中。（现在也支持直接复制链接了）
完成后，点击 "测试连接" 按钮。
![alt text](image-11.png)
成功后，会有提示。
点击 "保存设置" 按钮，完成配置。
恭喜你！你已经成功将Notion工作空间集成到我们的网站了。

一些使用说明：

- 每次添加条目后，刷新工作空间页面即可看到最新的条目。
- 你可以在Notion中编辑条目，工作空间会同步显示最新内容。
- 如果你更改了Notion集成秘钥或数据库ID，记得在工作空间设置中更新它们。
- 你可以自由编辑Select属性的选项，以适应你的分类需求。
- 描述会影响搜索结果的显示，建议填写有助于搜索的信息。
- 对于有填写账号密码的条目，工作空间的卡片左下角会显示绿色小点，方便识别，点击小眼睛可显示。
- 工作空间的内容只保存在本地浏览器缓存中，不会上传到任何服务器，请放心使用。
