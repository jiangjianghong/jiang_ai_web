注册与登录详解

账号安全与规则

我们的账号系统基于 Supabase 构建，提供企业级的安全认证。以下是详细的注册与使用规则：

注册限制规则

  • 用户名：必须在 2-20 个字符之间。支持中文、英文字母、数字、下划线 (_) 和短横线 (-)。不仅作为显示名称，也是您的唯一标识。
  • 邮箱验证：注册后通过正则表达式 /^[^\s@]+@[^\s@]+\.[^\s@]+$/ 严格校验格式。必须完成邮箱验证才能解锁云端同步功能。
  • 密码强度：最低要求 6 位字符。建议使用强密码以保护您的 Notion API 密钥等敏感数据。

卡片配置深度指南

卡片不仅仅是链接，它是高度可定制的入口。我们采用了智能抓取和多级缓存技术，确保最佳的加载速度和视觉体验。

参数详解

网址规范 (URL)

必须以 http:// 或 https:// 开头。系统会自动校验 URL 有效性。

标签系统 (Tags)

每个卡片最多支持 2个 标签。输入标签名后按回车添加。标签将极大提高搜索权重。

图标 (Favicon)

支持网络 URL 或本地上传图片（自动转为 Base64 或上传至云存储）。

自动获取技术 (Auto-Fetch)

点击"自动获取"时，系统会通过以下 5重策略 尝试获取最佳图标（按优先级）：

  1. Favicon.im 优化代理：通过代理加速访问，解决跨域问题。
  2. Google S2 服务：全球通用的图标抓取服务 (64px 高清)。
  3. DuckDuckGo Icons：备用图标源。
  4. 域名根目录：尝试直接访问 /favicon.ico。
  5. 本地默认：如果全部失败，使用系统默认图标兜底。

提示：系统会自动清除旧的图标缓存，确保您获取到网站最新的品牌标识。

智能权重算法

搜索本地卡片时，系统会基于复杂算法对结果进行排序，确保最相关的结果排在第一位：

  • 1. 完全或拼音匹配 150分
  • 2. 名称/首字母开头 130分
  • 3. 标签匹配 权重 x0.8
  • 4. 访问频率加权 Max +20分

极客功能 (Power User)

  • Visual Feedbacks：切换搜索引擎（Tab键）时会触发粒子烟花特效，不仅是视觉糖果，更是操作确认。
  • CORS 穿透建议：底层使用 JSONP 技术绕过浏览器跨域限制，直连百度 API 获取实时搜索热词搜索建议。
  • URL 智能识别：输入 localhost:3000 或 IP 地址时，系统会自动识别为直达链接而非搜索词。

隐私与数据架构

本地优先架构 (Local-First)

我们的核心架构基于 IndexedDB 和 LocalStorage。您的搜索历史、未登录状态下的卡片配置、UI 偏好设置等均完全存储在您的浏览器本地数据库中。

端到端数据流

当您连接 Notion 时，API 密钥仅保存在您的浏览器本地存储中，我们仅作为"客户端"直接与 Notion 官方 API 通信。您的数据流向为：浏览器 <-> Notion API，中间不经过我们的服务器中转。

完全的数据掌控权

导出：支持将所有配置一键导出为标准 JSON 格式文件，方便迁移或备份。
擦除："清除所有数据"功能会调用 localforage.clear() 和 localStorage.clear()，从物理层面彻底删除浏览器内的所有残留数据。
