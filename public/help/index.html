Notion 工作空间配置指南

通过集成 Notion，管理你的工作空间和收藏内容

概述

本指南将帮助你将 Notion 数据库集成到网站的工作空间功能中。通过简单的配置，你可以使用 Notion 作为后端数据库， 管理网站收藏、密码等信息，并享受实时同步的便利。

数据安全

所有数据存储在你的 Notion 账户中，本地仅缓存

实时同步

与 Notion 数据库保持实时同步，随时访问最新内容

灵活扩展

自定义字段和分类，适应不同的使用场景

演示效果

1登录 Notion

首先，你需要一个 Notion 账户。访问 notion.so 登录你的账户。如果还没有账户，可以免费注册。

提示：建议使用 Notion 桌面版应用，体验更流畅。该应用完全免费，请从官网下载。

2创建数据库

在 Notion 中创建一个新的数据库：

  1. 点击左侧边栏的"新增页面"按钮
  2. 选择"表格"模板
  3. 给数据库起一个名字，如"网站收藏"

5安装 Notion Web Clipper

为了方便地保存网页到 Notion 数据库，建议安装 Notion Web Clipper 扩展 。

  1. 在浏览器扩展商店搜索 "Notion Web Clipper"
  2. 点击"添加到浏览器"
  3. 登录你的 Notion 账户并授权

6添加第一个条目

使用 Web Clipper 保存网页：

  1. 访问任意网页
  2. 点击浏览器工具栏的 Notion 图标
  3. 选择刚创建的数据库
  4. 填写相关信息后保存
加载中...

7连接到工作空间

配置 API 密钥

在网站的工作空间设置中，粘贴之前复制的 Notion API 密钥。

配置数据库 ID

获取数据库 ID 的方法：

  1. 在 Notion 中打开你的数据库
  2. 复制浏览器地址栏的 URL
  3. URL 格式：https://notion.so/[数据库ID]?v=[视图ID]
  4. 提取其中的数据库 ID 部分
https://www.notion.so/abc123def456 ← 这部分是数据库 ID

测试连接

配置完成后，点击"测试连接"按钮验证配置是否正确。

配置完成！现在你可以在工作空间中管理你的 Notion 数据了。

使用技巧

  • 刷新页面即可同步最新的 Notion 数据
  • 可以在 Notion 中直接编辑条目，修改会实时同步
  • 使用分类标签可以更好地组织内容
  • 描述字段有助于搜索，建议填写关键词
  • 包含账号密码的条目会有特殊标记

隐私说明

数据安全：工作空间的所有数据仅存储在你的浏览器本地缓存中， 不会上传到我们的服务器。API 密钥等敏感信息均采用加密存储。

权限说明：网站仅读取你授权的 Notion 数据库， 不会访问你的其他 Notion 内容。
